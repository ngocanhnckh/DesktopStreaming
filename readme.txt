Desktop Presenter: Streamer client
wbserver: Receiever
wbclient: WebClient written on sivlerlight
The Desktop Presenter will stream it's video to wbserver. Wbserver act as a chatroom host, receiving every packets sending from different clients. So that you can view many streamers in one chatroom, that means many Desktop Presenter can present to one server at the same time.
The final goal is to make the video drawing viewable on web browser using javascript
